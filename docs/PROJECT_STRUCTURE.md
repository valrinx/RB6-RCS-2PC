# Project Structure

- `rvn.py` - main FastAPI app, controller logic, and built-in tests
- `mouse/` - hardware adapter helpers
- `templates/` - HTML template
- `static/` - frontend JS/CSS assets
- `configs/` - default/sample profile JSON files
- `scripts/` - local utility scripts (e.g. build helpers)
- `.github/workflows/` - CI and release automation
- `docs/` - project documentation and contributor notes
